CA116 Computer Science Module
Welcome to the CA116 Computer Science module repository!

About
This repository contains materials and projects related to my CA116 module, which is part of my computer science degree program. CA116 is focused on Computer Programming 1, and all the contents here are related to Python programming.

Contents
Lecture Notes: Detailed notes from lectures covering fundamental concepts and topics in Python programming.
Assignments: Programming assignments and projects given throughout the module.
Practice Exercises: Additional exercises for practice and reinforcement of Python programming concepts.
Resources: Useful resources such as recommended textbooks, online tutorials, and programming tools.
